Build a Python command-line tax calculator in the sandbox. It computes US
federal income tax for a given annual gross income and filing status.

Requirements:
- Filing status: "single" or "married_joint".
- 2023 progressive brackets:
  * single: 10% <=11000, 12% <=44725, 22% <=95375, 24% <=182100, 32% <=231250, 35% <=578125, 37% above
  * married_joint: 10% <=22000, 12% <=89450, 22% <=190750, 24% <=364200, 32% <=462500, 35% <=693750, 37% above
- Standard deduction: single 13850, married_joint 27700. Taxable income = max(0, gross - standard deduction).
- Apply brackets to taxable income (marginal/bracket computation).
- Optional long-term capital gain (argument --gain): taxed at 0/15/20% using the
  brackets based on taxable income (0% if taxable <= 44625 single / 89350 married,
  15% if <= 492300 single / 553850 married, else 20%). Add it to total tax.
- Print an itemized breakdown: gross, standard deduction, taxable income,
  income tax, capital gains tax, total tax, effective rate (total/gross).
- Validate inputs: non-negative numbers, valid filing status; print a clear
  error otherwise.

Write the code, then TEST it with several cases (e.g. single gross=50000,
married_joint gross=150000 with gain=20000, and an edge case of gross=0).
Verify the numbers are sensible. When the goal is met and verified, reply exactly: DONE.
